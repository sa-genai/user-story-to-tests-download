# Adding Download Button

## Front end
- Need to add "Download" Buttons.

## Sample prompt for UI
```
1. Read my entire workspace and understand how UI implemented in the frontend folder
2. I want to add buttons to download  the generated testcases in different formats such as json, csv and excel
3. Implement the code for the above added buttons. and make sure they are working
```
## Steps to implement
1. Unzip and Open the folder in vscode
2. Install node modules ```npm install```
3. Add ```api-key``` in the ```.env``` file.
4. Start the server ```npm run dev```.
5. Open the brower and launch ```localhost:5173``` . Make sure UI is running.
6. Ask co-pilot to add download button ```provide prompts```
7. Stop the server and start again. and check for download buttons in UI.
8. Ask co-pilot to implemnet code for the button. Accept proposed changes.
9. Stop the server and start again. 
10. Generate test cases and check the buttons's functionality.
